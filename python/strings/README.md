### Python Strings

| Challenge | Description |
| :--- | ---: |
| lowercase | Write a function called lowercase that, given a string, returns the entire sting as lowercase | 
| Uppercase | Write a function called uppercase that, given a string, returns the entire sting as uppercase |
| Switch case | Write a function called switch_case, given a string, returns the string with uppercase letters in lower case and vice-versa. Include punctuation and non-cased characters unchanged | 
| aardvark zebra index | Write a function called aardvark, given a string returns 'aardvark' if the string starts with 'a'. Otherwise returns zebra |
